=================================
rubanjs - Fractal JavaScript API
=================================

rubanjs is a collection of libraries which allow you to interact with a local or remote Fractal node,
using an HTTP, WebSocket or IPC connection.

The following documentation will guide you through :ref:`installing and running ruban.js <adding-ruban>`,
as well as providing a :ref:`API reference documentation <#id1>` with examples.
